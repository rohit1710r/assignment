HTML Tag lists

  1. head tag : The head tag in HTML is used to define the head portion of the document which contains information related to the document.
  2. body tag : The body tag in HTML is used to define the main content present inside an HTML page.
  3. title tag : The title tag in HTML is used to define the title of HTML document. It sets the title in the browser toolbar.
  4. image tag : HTML Image, how to add the image in HTML. In earlier times, the web pages only contains textual content, which made them appear quite boring and uninteresting.
  5. list tag: The list tag in HTML is used to define the list item in an HTML document. It is used within an Ordered List <ol> or Unordered List <ul>.
  6. table tag : HTML Table, various ways to implement it, & will also understand its usage through the examples. HTML Table is an arrangement of data in rows and columns, or possibly in a more complex structure.

  9. div tag : The div tag is used in HTML to make divisions of content in the web page (text, images, header, footer, navigation bar, etc).
